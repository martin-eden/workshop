## Itness codec

Itness is a strings tree format

Actually it's parenthesized tree representation with one-level
directed quotes (`[` `]`).

Format traits:

  * Tree structure
  * Free whitespaces
  * No data restrictions

Comparison with other tree formats:

| Format | Comments | Free whitespaces | Named values | Typed values |
|-------:|:--------:|:----------------:|:------------:|:------------:|
| Is     |    ☐     |        ☑         |      ☐       |      ☐       |
| JSON   |    ☐     |        ☑         |      ☑       |      ☑       |
| YAML   |    ☑     |        ☐         |      ☑       |      ☑       |
